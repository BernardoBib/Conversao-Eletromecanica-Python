# Conversão Eletromecânica em Python

Este repositório apresenta a resolução computacional de três questões da disciplina de **Conversão Eletromecânica**, utilizando **Python** como ferramenta de apoio para modelagem, cálculos numéricos e geração de gráficos.

## Objetivo

O projeto tem como objetivo auxiliar na análise e compreensão de problemas clássicos da disciplina, por meio da implementação de códigos que automatizam cálculos, organizam resultados e facilitam a visualização do comportamento das grandezas envolvidas.

## Conteúdo do repositório

- `Q1.ipynb` — resolução da Questão 1
- `Q2.ipynb` — resolução da Questão 2
- `Q3.ipynb` — resolução da Questão 3

## Ferramentas utilizadas

- Python
- NumPy
- Matplotlib
- SciPy
- SymPy
- Jupyter Notebook

## Aplicações desenvolvidas

Neste projeto, os códigos foram elaborados para:

- realizar cálculos relacionados aos problemas propostos;
- analisar grandezas eletromecânicas envolvidas em cada questão;
- gerar gráficos para apoio à interpretação dos resultados;
- organizar a resolução de forma computacional e reprodutível.

## Contexto acadêmico

Este projeto foi desenvolvido como parte das atividades da disciplina de **Conversão Eletromecânica**, com foco no uso de ferramentas computacionais como suporte ao estudo e à resolução de exercícios da área.
